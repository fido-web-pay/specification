FIDO® Web Pay (FWP) - Introduction
EMV® represents the current "Gold Standard" for secure and convenient payments in the physical world.
This document describes a Web-adapted version of EMV, building on FIDO2 technology combined with W3C's PaymentRequest API.
However, unlike EMV, FWP is intended to be applicable to any account-based payment system, including the eurozone's SEPA Inst.
Also in contrast to EMV, FWP encrypts user-authorizations, which has the following side effects:
  • Eliminating dependencies on external tokenization services.
  • Reducing the amount of personally identifiable information (PII), shared with third-parties to the name of the payer's bank.
  • Eliminating PCI requirents with respect to the payer.
Although not a part of this specification, the underpinning authorization scheme also opens the door to non-direct payments, including secure (tokenized) versions of Card-on-File.
